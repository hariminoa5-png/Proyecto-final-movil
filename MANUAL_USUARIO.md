# Manual de Usuario - AutoTrack

¡Bienvenido a AutoTrack! Esta guía te ayudará a configurar y utilizar tu aplicación de gestión vehicular.

## 1. Inicio de Sesión
Si ya tienes una cuenta activa, simplemente ingresa tu **matrícula** y tu **contraseña**. 

- **¿Olvidaste tu clave?**: Presiona el enlace en la parte inferior, ingresa tu matrícula y el sistema te asignará una clave temporal (normalmente `123456`) para que puedas volver a entrar.

## 2. Registro de Cuenta Nueva
Para empezar a usar AutoTrack, debes completar dos pasos:

### Paso 1: Registro (Obtener Token)
1. Ve a la pantalla de **Crear Cuenta**.
2. Ingresa tu **Matrícula ITLA**.
3. Presiona **Enviar**. Si tu matrícula es válida, el sistema te mostrará un mensaje de éxito y te llevará a la siguiente pantalla automática.

### Paso 2: Activación
1. El sistema cargará el token que recibiste automáticamente.
2. Crea una **Nueva Contraseña** (mínimo 6 caracteres).
3. Confirma la contraseña.
4. Presiona **Activar Cuenta**. ¡Listo! Ahora serás redirigido al Dashboard principal.

## 3. Uso del Dashboard
En la pantalla principal (Inicio) encontrarás:

- **Slider de Consejos**: Información útil sobre el mantenimiento de tu vehículo.
- **Frases Motivacionales**: Mensajes positivos que cambian aleatoriamente.
- **Acceso Rápido**: Iconos para gestionar tus Vehículos, Mantenimiento, Combustible, Gomas y más.
- **Barra de Navegación**: Cambia fácilmente entre Inicio, Noticias, Vehículos, Foro y tu Perfil.

## 4. Cerrar Sesión
Para salir de tu cuenta, presiona el icono de perfil en la parte superior derecha de la pantalla de Inicio.
